# Smart Calculator

A Python-based Smart Calculator built using functions and mathematical operations. This project demonstrates the use of Python functions, parameters, return statements, conditional statements, and built-in functions.

## Features

This Smart Calculator can perform:

- Addition
- Subtraction
- Multiplication
- Division
- Square calculation
- Cube calculation
- Largest number finder
- Smallest number finder
- Even or Odd checker
- Student Grade calculation

## Concepts Used

- Functions
- Parameters
- Return Statements
- Conditional Statements (if, elif, else)
- Mathematical Operations
- Built-in Functions (max and min)
- Problem Solving Skills

## Technologies Used

- Python 3
